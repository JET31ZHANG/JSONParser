# JSONParser
一个个人学习用的 JSON 解析器项目

简介
===

功能
- 解析 JSON 支持的各种数据格式（字符串、数字（整数和小数，支持负数）、布尔值、null）
- 解析 JSON 支持的数据结构（包括数组和对象，支持嵌套）
